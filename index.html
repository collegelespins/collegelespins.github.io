Les autres sites

Ce site est écrit (à la main) en html et en typescript.
Il fait donc appel à des fonctions compilées en javascript.

Il est destiné à la production de code html, de fichiers .pdf ou de documents .docx ou .pptx à partir de simples données .txt ou .xml.

L'écriture du site avec Visual Studio Code et le suivi de données Github permettent de se passer d'un autre type de transfert et d'actualisation.
